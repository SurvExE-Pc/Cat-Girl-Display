HTML for this demo:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Demo</title>
    <script src="https://ashmi.ga/Cat-Girl-Display/cat-girl-counter.min.js"></script>
</head>
<body>
    <div id="counter"></div>
    <script>
        console.log("Loaded body script");
        const placeHolder = document.createElement('cat-girl-counter');
        document.getElementById("counter").appendChild(placeHolder);
        placeHolder.count = Math.floor(Math.random() * (100 - 1 + 1)) + 1;  // This just randomly generates a number from 1 - 100.
    </script>
</body></html>


To use this put the script tag at the bottom of the head element without any tags like async or defer,
Like in the example :D!
